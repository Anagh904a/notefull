## ✨ Features

### 🔐 **Privacy & Security**
- **Local Storage** - Your notes stay on your device
- **Password Protection** - Lock sensitive notes
- **Shield Scan** - Built-in security scanning for sensitive data
- **No Account Required** - No sign-ups, no tracking

### 📝 **Notes**
- Create and manage text notes
- Quick Search - Find any note instantly
- Manage multiple notes easily 

### ✅ **Lists**
- Create checklists and to-do lists
- Track progress with visual indicators
- Manage multiple lists easily

### 💾 **Storage**
- **LocalForage** - Fast, reliable local storage (50MB+)
- **Sandboxed** - Data isolated per domain for security
- **Data Recovery** - Built-in recovery system

### 🎨 **User Experience**
- Beautiful, clean interface
- Responsive design - works on all devices
- Fast & lightweight
- Intuitive and easy to use

## 🚀 Getting Started

### 📱 Mobile App
Download Notefull on the Play Store and start taking notes immediately!

[📥 Get it on Google Play](https://play.google.com/store/apps/details?id=app.notefull.com)

### 🌐 Web Version
Visit our [live demo](https://notefull.netlify.app) to try Notefull in your browser.

### Self-Host
1. Clone the repository
```bash
git clone https://github.com/Anagh904a/notefull.git
```

2. Open in browser
```bash
cd notefull
# Simply open index.html in your browser
# Or use a local server:
python -m http.server 8000
```

3. Start taking notes! 📝

## 🛡️ Privacy Commitment

Notefull is built with privacy as the foundation:

- ✅ **No data collection** - We don't collect, store, or transmit your data
- ✅ **No analytics** - No tracking, no telemetry, no fingerprinting
- ✅ **No servers** - Everything runs locally in your browser
- ✅ **No accounts** - No email, no password, no personal information
- ✅ **Open source** - Full transparency, audit the code yourself
